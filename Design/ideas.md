# GMLB Automation — Идеи дизайна

## Концепция 1

<response>
<text>
**Design Movement:** Технологический минимализм с элементами Brutalist Tech
**Core Principles:**
- Острые контрасты: чёрный фон #0A0A0F, белый текст, ярко-синие акценты
- Монументальная типографика: огромные заголовки с tracking-tight
- Сетка с намеренными нарушениями — элементы, выходящие за рамки
- Данные как визуальный элемент: числа, метрики, коды — часть декора

**Color Philosophy:** Почти чёрный фон (#0A0A0F), чисто белый текст, электрический синий (#2563EB) как единственный акцент. Ощущение терминала, машинного кода.

**Layout Paradigm:** Асимметричная сетка. Заголовки Hero занимают 70% ширины, карточки продуктов — разные размеры (featured + mini). Секции разделены тонкими горизонтальными линиями.

**Signature Elements:**
- Мерцающий курсор в заголовках (typing effect)
- Тонкие grid-линии на фоне (как инженерная бумага)
- Числовые счётчики с анимацией

**Interaction Philosophy:** Hover — подсветка синим контуром. Клик — мгновенная реакция без задержек.

**Animation:** Текст появляется построчно (stagger), числа считаются вверх при входе в viewport.

**Typography System:** Space Grotesk (заголовки, bold/black) + JetBrains Mono (акценты, коды) + Inter (тело).
</text>

## Концепция 2

<response>
<text>
**Design Movement:** Dark Glassmorphism / Vercel-inspired Premium SaaS
**Core Principles:**
- Глубокий тёмный фон с тонкими градиентами (#0D1117 → #0A0F1E)
- Glassmorphism карточки: backdrop-blur, полупрозрачные границы
- Синий/голубой неоновый акцент с мягким свечением
- Плавные анимации появления (fadeInUp, staggered)

**Color Philosophy:** Фон — почти чёрный с синеватым оттенком. Акценты — #3B82F6 (синий) и #60A5FA (голубой). Вторичный — фиолетовый #8B5CF6 для градиентов. Ощущение ночного неба над технологичным городом.

**Layout Paradigm:** Центрированный контент с максимальной шириной 1280px. Hero — полноэкранный с анимированным градиентным фоном. Продукты — сетка 3 колонки на desktop.

**Signature Elements:**
- Анимированные градиентные сферы/блобы на фоне Hero
- Glassmorphism карточки с hover-подъёмом
- Градиентные кнопки синий→фиолетовый

**Interaction Philosophy:** Плавные переходы 300ms. Hover на карточках — подъём + усиление свечения.

**Animation:** Framer Motion fadeInUp при скролле. Параллакс на Hero.

**Typography System:** Geist (заголовки) + Inter (тело). Размеры: H1 72px, H2 48px, H3 32px.

## Концепция 3

<response>
<text>
**Design Movement:** Cyberpunk Industrial — тёмный, агрессивный, технологичный
**Core Principles:**
- Фон #080C14 (почти чёрный с синим), акцент — неоново-голубой #00D4FF
- Угловые срезы (clip-path) на секциях вместо прямых границ
- Текстурированный фон: шум/grain поверх градиентов
- Жёсткая типографика: condensed bold шрифты

**Color Philosophy:** Тёмно-синий фон, неоново-голубой акцент #00D4FF, вторичный — пурпурный #FF00FF для редких акцентов. Ощущение киберпанка, Blade Runner.

**Layout Paradigm:** Диагональные секции через clip-path. Карточки с угловыми срезами. Асимметричный Hero с изображением справа.

**Signature Elements:**
- Диагональные разделители секций
- Scanline-эффект на изображениях
- Неоновые рамки на hover

**Interaction Philosophy:** Агрессивные hover-эффекты: цвет меняется мгновенно, свечение усиливается.

**Animation:** Быстрые переходы 150ms. Glitch-эффект на заголовках при hover.

**Typography System:** Orbitron (заголовки) + Rajdhani (подзаголовки) + Inter (тело).
</text>
